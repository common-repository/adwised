== Description ==

This plugin is a free web push notification service for WordPress websites.
you can add web push notifications to any website via Adwised.
This platform is available for Persian/Farsi language ( We are working on English version)

Web push notifications are notifications that can be sent to a user via desktop web and mobile web.
Web push notifications are delivered on a user's desktop or mobile screen anytime when they are online.
To use this plugin, you should have an account in [adwised.com](adwised.com)'s console in [panel.adwised.com](http://panel.adwised.com)

== Installation ==

= FROM YOUR WORDPRESS DASHBOARD =

- Visit **Plugins** > **Add New**
- Search for **adwised**
- Install & activate **adwised** from your Plugins page.
- Now you can find adwised under the left menu.
- Sign in/Sign up in [panel.adwised.com](https://panel.adwised.com) and create a website
- Get your script from [panel.adwised.com](https://panel.adwised.com) and enter it in settings page
